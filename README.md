# E-Commerce Frontend

## 🚀 Description

Frontend for a full-stack e-commerce application built with Next.js.

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS

## 🔐 Features

- Authentication (Login/Register)
- Protected routes
- Product listing
- Cart system
- Checkout flow
- Orders history

## ⚙️ Environment Variables

NEXT_PUBLIC_API_URL=https://your-backend.onrender.com/api

## ▶️ Run locally

npm install
npm run dev
